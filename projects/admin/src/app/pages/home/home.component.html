Welcome to Novus App

An Angular 21 Admin Dashboard Template by Novus Apps

Technology Stack

  • Angular 21 — Signal-based change detection, standalone components
  • Angular Material 21 — MDC-based component library
  • Tailwind CSS 4+ — Utility-first layout and bespoke styling
  • TypeScript — Strict type-safe development
  • Firebase 11+ — Auth, Firestore, Cloud Functions
  • Tabler Icons — Clean, consistent icon library
  • ApexCharts — Interactive dashboard charting
  • Playwright — E2E testing across both apps

Key Features

  • Monorepo — Admin and User apps sharing one workspace
  • Signal-First State — Reactive UI without RxJS boilerplate
  • Offline-First Data — LocalStorage → Firestore sync via DataShareService
  • Firebase Emulators — Zero-cost local development environment
  • Responsive Layout — Desktop, Tablet, and Mobile breakpoints
  • Route Guards — Auth-protected admin routes
  • Splash Sequence — Branded initialization flow
  • CI/CD Ready — GitHub Actions → Firebase Hosting

Getting Started

  • Add new pages by creating routes in pages.routes.ts
  • Extend navigation in sidebar-data.ts
  • Customize the theme in src/assets/scss/
  • Configure Firebase in environments/environment.local.ts
