# USR-501: User App Home Page

**Epic:** USR-500 User Application  
**Status:** ✅ Done  
**Priority:** P1

---

## Story

> As an end-user, I want to land on a clean, informative home page after the splash screen so that I understand the application's purpose and can navigate to my profile or key sections.

---

## Acceptance Criteria

- [x] A `/dashboard/home` route exists in `pages.routes.ts` and is the default redirect after splash.
- [x] `HomeComponent` renders with the `FullComponent` layout (sidebar + header).
- [x] The page shows a welcome message and a brief description of the app.
- [x] A prominent **"My Profile"** call-to-action navigates to `/dashboard/profile` (USR-502).
- [x] The page is responsive across Mobile, Tablet, and Desktop breakpoints.
- [x] `data-test-id="home-welcome-message"` and `data-test-id="home-profile-cta"` are present.
