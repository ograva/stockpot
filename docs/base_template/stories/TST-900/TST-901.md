# TST-901: Playwright Multi-App Infrastructure

## Story Description
As a QA engineer, I want an automated testing suite that covers both the Admin and User apps so that I can detect regressions across the entire monorepo.

## Acceptance Criteria
- [ ] `playwright.config.ts` handles tests for Port 4200 (Admin) and Port 4400 (User).
- [ ] A sample test `T001: Login Smoke Test` passes for both applications.
- [ ] Reports generated correctly in GitHub Actions CI/CD.
- [ ] Standardized `data-test-id` usage documented.
