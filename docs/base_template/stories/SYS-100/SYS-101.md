# SYS-101: Initialize Firebase Project

## Story Description
As a developer, I want the Firebase Auth and Firestore layers to be initialized so that I have a secure and persistent backend for the template.

## Acceptance Criteria
- [ ] Firebase Emulators are configured for local development.
- [ ] `firestore.rules` blocks all writes by default except for authenticated users to their own documents.
- [ ] A `settings` collection exists with baseline global site configuration.
- [ ] Auth enabled with Email/Password protocol.
- [ ] Automated deployment configured via GitHub Actions.
