# SYS-102: Integrate Tailwind CSS Hybrid Styling

## Story Description

As a developer, I want Tailwind CSS v4+ integrated into the Angular workspace so that I can rapidly build bespoke layouts without writing repetitive custom SCSS, while Material UI continues to handle complex component behavior.

## Technical Approach

- Install `tailwindcss` and `@tailwindcss/postcss` as dev dependencies.
- Create a root-level `postcss.config.mjs` pointing to the `@tailwindcss/postcss` plugin.
- Add `@import "tailwindcss"` to both app global stylesheets.
- Define a Tailwind CSS theme block that maps Tailwind color tokens to Material CSS Custom Properties (e.g., `var(--mat-sys-primary)`), ensuring zero "CSS Drift."

## Acceptance Criteria

- [ ] Tailwind utility classes work in both `projects/admin` and `projects/user-app`.
- [ ] Tailwind color palette is synchronized with Material theme CSS variables.
- [ ] No conflicts between Tailwind's `preflight` reset and Material component styles.
- [ ] Build output shows no Tailwind/PostCSS errors.
